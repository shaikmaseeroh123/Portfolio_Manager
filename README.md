# Portfolio_Manager
Portfolio is a simple and user-friendly website to manage projects and tasks. It shows your projects, progress, recent activities, and upcoming work in one place. The design is clean and easy to understand, helping you stay organized, track deadlines, and manage your work smoothly without confusion.

# Portfolio Manager

A modern, professional project and task management application built with vanilla HTML5, CSS3, and JavaScript.

## Features

- **Dashboard Overview**: Stats, recent projects, activity feed, and upcoming tasks
- **Project Management**: Create, edit, delete projects with custom colors and categories
- **Task Management**: Kanban board with drag-and-drop functionality
- **Analytics**: Visual charts showing project progress and task distribution
- **Dark/Light Theme**: Toggle between themes with persistent preference
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Local Storage**: All data persists in browser storage
